Linux大作业

Posted on Dec 17, 2022

大作业完成步骤

1.安装git

apt install git

2.git配置

设置git提交电子邮件和用户名

root@debian:~# git config --global user.name "jdiosj"
root@debian:~# git config --global user.email "1299844731@qq.com"

验证配置信息

root@debian:~# git config --list
user.name=jdiosj
user.email=1299844731@qq.com

生成ssh key：

ssh-keygen -t rsa -C "1299844731@qq.com"

查看 id_rsq.pub 文件

root@debian:~/.ssh# cat id_rsa.pub

将以上内容复制到GitHub网站
在github上创建仓库

3.安装与配置hugo
安装hugo

root@debian:~# wget https://github.com/gohugoio/hugo/releases/download/v0.88.1/hugo_0.88.1_Linux-64bit.tar.gz

加压压缩包

root@debian:~# tar -zxvf hugo_0.88.1_Linux-64bit.tar.gz -C /usr/local/bin

创建博客位置

生成主题
修改配置文件

root@debian:~# cd /usr/local/bin/myblog/
root@debian:/usr/local/bin/myblog# ls
archetypes  client    config.toml  data  layouts  resources  server.c  themes
blog	    client.c  content	   docs  public   server     static
root@debian:/usr/local/bin/myblog# vim config.toml

创建博客

root@debian:/usr/local/bin/myblog# hugo new post/first-blog.md

使用Markdown语法编写博客

将相关的图片放入static文件夹中 

root@debian:/usr/local/bin/myblog/content/post# vim first-blog.md

访问站点

root@debian:/usr/local/bin/myblog# hugo server

生成静态网页

root@debian:hugo
